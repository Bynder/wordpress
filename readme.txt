== Description ==

= Embed your WebDAM assets easily in your own WordPress application =

The WebDAM Asset Chooser for WordPress allows you to embed assets from your WebDAM account directly into WordPress pages or posts.

== Installation ==

To run this plugin, simply install the scripts and then:
1. Visit Settings > WebDAM
2. Enter your WebDAM Domain
3. Follow the API prompts to authenticate with WebDAM

And that's it, you are all set. You can now embed assets by clicking on the new WebDAM icon when using the WordPress editor.
