== Description ==

= Embed your WebDAM assets easily in your own Wordpress application =

The WebDAM Asset Chooser for Wordpress allows you to embed assets from your WebDAM account directly into Wordpress pages or posts. Simply enter the domain that you integrate with in the WebDAM settings and you are all set.


== Installation ==

To run this plugin, simply install the scripts and then:
1. Go to the new 'WebDAM' link under admin settings (see screenshot 2)
2. Enter the domain that you wish to link to.

And that's it, you are all set. You can now embed assets by clicking on the new WebDAM icon when using the Wordpress editor.
